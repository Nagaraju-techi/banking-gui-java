# Banking GUI Java Project

A Java-based **Banking Management System** with Graphical User Interface (GUI) developed using **Java Swing**. This project allows users to register, log in, manage account transactions, and view transaction history with file-based data storage.

---

## Features

- User Registration  
- Secure Login Authentication  
- Deposit Money  
- Withdraw Money  
- Check Account Balance  
- View Transaction History  
- Logout Option  
- File Handling for Data Storage (`users.txt`)  
- Simple and User-Friendly GUI  

---

## Technologies Used

- Java  
- Java Swing  
- AWT  
- OOP Concepts  
- File Handling  
- Collections Framework (`HashMap`, `ArrayList`)
